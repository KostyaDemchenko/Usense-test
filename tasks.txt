Task:
Create applications to test password strength.

The essence of the assignment:
Create a field for entering a password, under the field add 3 sections which will show the strength of the password;
Changes in password strength must take place in real time;
How to calculate the strength of a password:
Only letters/digits/symbols - the password is easy;
Combination of letters-symbols/letters-digits/digits-symbols - the password is medium;
Has letters, symbols and numbers - the password is strong;
The color of the sections will depend on the strength of the password:
If the field is empty, all sections are gray;
If the field has less than 8 characters, all sections are red;
If the password is easy - the first section is red the rest are gray;
If the password is medium - the first two sections are yellow the last one is gray;
If the password is strong, all sections are green;




Задача:
Создавайте приложения для проверки надежности пароля.

Суть задания:
Создайте поле для ввода пароля, под полем добавьте 3 раздела, которые будут показывать надежность пароля;
Изменения надежности пароля должны происходить в режиме реального времени;
Как рассчитать надежность пароля:
Только буквы/цифры/символы - пароль простой;
Комбинация букв-символов/букв-цифр/цифр-символов - пароль средний;
Имеет буквы, символы и цифры - пароль надежный;
Цвет разделов будет зависеть от надежности пароля:
Если поле пусто, все разделы отображаются серым цветом;
Если в поле меньше 8 символов, все разделы окрашены в красный цвет;
Если пароль простой - первый раздел красный, остальные серые;
Если пароль средний - первые два раздела желтые, последний серый;
Если пароль надежный, все разделы отображаются зеленым цветом;
